### Overview

Low-Ops allow you to manage `Users` from `Administration` interface.

You can access the `Users` by navigating to `Administration > Users`.

<img src="/img/admin-users.png" alt="Administration Users" className="image" />

### Create User

To create a new user, click on the `Create User` button and fill the form.

<img src="/img/create-user-dialog.png" alt="Create User Dialog" className="image" />

### User Details

To view user details, click on the user name on the user table. You can `Reset Password` or view `User Teams` at this
screen.

<img src="/img/user-details.png" alt="User Details" className="image" />

### Delete User

To delete a user, click on the `Delete` button on the down of user details page.

:::tip Best Practices

- Use a strong, unique password for each new user
- Inform the new user to change their password upon first login
- Assign appropriate permissions to the new user based on their role
- Regularly review and update user lists to maintain security

:::
