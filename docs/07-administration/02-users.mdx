---
title: Users
---

Manage users through the `Administration` interface.

Go to `Administration > Users` to access user management.

<img src="/img/admin-users.png" alt="Administration Users" className="image" />

### Create User

Click the `Create User` button and fill out the form to add a new user.

<img src="/img/create-user-dialog.png" alt="Create User Dialog" className="image" />

### User Details

Click a username in the table to view user details. Here you can `Reset Password` or check `User Teams`.

<img src="/img/user-details.png" alt="User Details" className="image" />

### Delete User

Click the `Delete` button at the bottom of the user details page to remove a user.

:::tip Best Practices

- Use strong, unique passwords for each user
- Ask users to change their password on first login
- Assign permissions based on user roles
- Regularly review and update user access

:::
