### Overview

Applications are the core of the platform. When you open the Low-Ops you will see the applications you have access to.

<img src="/img/applications.png" alt="Administration Applications" className="image" />

### Create Application

Applications can be create from 2 places:

- From the `Create Application` button in the `applications/root` screen.
- From the `Administration > Applications` screen.

Select the `Template` you want to use to create the application and input the `Name` and `Description`. If you can't see
the template you want to use, you can create a new template from the `Administration > Templates` screen.

<img src="/img/create-app-1.png" alt="Create Application 1" className="image" />
<img src="/img/create-app-2.png" alt="Create Application 2" className="image" />

### Application Details

To view the application details, click on the application name in the applications table. Navigate to the application
settings screen. `Application > Settings`. You can change the description of the application on that screen.

<img src="/img/app-settings.png" alt="Application Settings" className="image" />

### Delete Application

Applications can be create from 2 places:

- From the `Application > Settings` screen. Bottom of the screen have `Delete Application` button.
- From the `Administration > Applications` screen. From the `Delete` button in the application table menu.
