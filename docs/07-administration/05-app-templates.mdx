---
title: Templates
---

Templates help you create applications by defining their structure. Low-Ops provides predefined templates, and you can
also create custom ones.

<img src="/img/templates-screen.png" alt="Templates Screen" className="image" />

### Create Template

Go to `Administration > Templates` to create a new template. Click `Upload Template` button and fill in the required
fields.

:::note

Choose the correct template type (e.g., `Mendix` for Mendix applications, `NextJS` for NextJS applications). Use
`Generic` type if no specific type matches your template.

:::

<img src="/img/upload-template.png" alt="Upload Template" className="image" />

### Delete Template

To delete a template, go to `Administration > Templates` and click the `Delete` button in the template table menu.
